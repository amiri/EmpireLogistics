# Description

Installs the [Geometry Engine, Open Source (GEOS)](http://trac.osgeo.org/geos/) library.

## Platforms:

Tested on:

* Ubuntu 12.04

## Recipes:

* `default.rb` - Downloads, builds, and installs GEOS from source.

## Attributes

* `node['geos']['version']` - the version of GEOS to use.
